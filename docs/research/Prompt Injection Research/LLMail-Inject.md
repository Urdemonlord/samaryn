# LLMail-Inject

Full title: *LLMail-Inject: A Dataset from a Realistic Adaptive Prompt Injection Challenge*

Link: https://arxiv.org/abs/2506.09956

## Fokus

Dataset realistis dan adaptif untuk [[Indirect Prompt Injection]] dalam skenario email assistant.

## Kenapa penting

- paling kuat jika fokusmu adalah [[Prompt Injection Dataset]]
- berasal dari challenge publik realistis
- menangkap perilaku attacker yang adaptif

## Detail utama

- 208,095 unique attack submissions
- 839 participants
- data dan challenge code dirilis publik
