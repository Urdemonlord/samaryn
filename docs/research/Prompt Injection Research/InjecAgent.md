# InjecAgent

Full title: *Benchmarking Indirect Prompt Injections in Tool-Integrated Large Language Model Agents*

Link: https://arxiv.org/abs/2403.02691

## Fokus

Benchmark untuk [[Prompt Injection Agent Benchmark]] pada agent LLM yang memakai tools.

## Kenapa penting

- fokus pada agentic setting
- relevan untuk email, web, tools, dan action-taking agents
- spesifik untuk [[Indirect Prompt Injection]]

## Detail utama

- 1,054 test cases
- 17 user tools
- 62 attacker tools
- dua niat utama serangan: direct harm dan data exfiltration
