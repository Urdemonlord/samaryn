# MPIB

Full title: *MPIB: A Benchmark for Medical Prompt Injection Attacks and Clinical Safety in LLMs*

Link: https://arxiv.org/abs/2602.06268

## Fokus

Benchmark domain medis untuk [[Medical Prompt Injection]] pada LLM dan RAG.

## Kenapa penting

- fokus domain klinis
- mengukur safety, bukan cuma kepatuhan terhadap serangan
- relevan untuk use case medis berisiko tinggi

## Detail utama

- 9,697 curated instances
- memakai CHER dan ASR
- mencakup direct injection dan indirect RAG-mediated injection
