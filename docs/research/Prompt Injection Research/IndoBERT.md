# IndoBERT

IndoBERT adalah keluarga model BERT untuk bahasa Indonesia yang relevan untuk task klasifikasi teks, termasuk deteksi prompt berbahaya dan [[Out-of-Domain Detection]].

## Peran dalam skripsi

- engine deteksi out-of-domain prompt injection
- classifier sebelum request diteruskan ke LLM
- komponen ML di dalam [[Samaryn AI Security Gateway]]
