# Out-of-Domain Detection

Out-of-domain detection digunakan untuk mengenali input yang menyimpang dari distribusi data yang diharapkan, sehingga berguna untuk menyaring prompt berbahaya, anomali, atau request yang tidak sesuai konteks.

## Relevansi ke skripsi

- inti untuk engine deteksi berbasis [[IndoBERT]]
- dapat dipakai sebagai sinyal awal sebelum evaluasi keamanan lebih lanjut
- mendukung arsitektur berlapis pada [[Samaryn AI Security Gateway]]
