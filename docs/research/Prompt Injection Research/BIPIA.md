# BIPIA

Full title: *Benchmarking and Defending Against Indirect Prompt Injection Attacks on Large Language Models*

Link: https://arxiv.org/abs/2312.14197

## Fokus

Benchmark foundational untuk [[Indirect Prompt Injection]] pada LLM yang membaca konten eksternal.

## Kenapa penting

- benchmark awal yang terstruktur
- kuat untuk [[Prompt Injection Benchmark]]
- cocok untuk evaluasi lintas task

## Detail utama

- 5 task: email QA, web QA, table QA, summarization, code QA
- total sekitar 626,250 train prompts dan 86,250 test prompts
- evaluasi attack dan defense
